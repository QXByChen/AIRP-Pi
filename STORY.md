# 故事驱动技能 — 让角色扮演引人入胜

基于叙事学六大理论体系（亚里士多德诗学、波尔蒂36种戏剧模式、坎贝尔英雄之旅、麦基《故事》、救猫咪节拍表、ATU民间故事类型学）提炼。

## 核心原则

### 1. 价值转换（每个场景的铁律）
每个场景结束时必须有某种价值发生逆转——希望→绝望、信任→背叛、亲近→疏离。如果场景结束时情感状态与开始时相同，这个场景就是多余的。在角色扮演中，每轮回复至少推动一个微小的价值变化。

### 2. 信息不对称（保持悬念的核心引擎）
三种配置灵活切换：
- **观众知道 > 角色知道**：戏剧性反讽，制造"炸弹在桌下"的紧张
- **角色知道 > 观众知道**：神秘叙事，制造好奇心
- **所有人都不完全知道**：悬疑叙事，制造不确定性

螺旋式揭示：信息逐层释放，每一层既回答旧问题又引出新问题。

### 3. 打破预期（熟悉 + 新颖 = 最佳体验）
- 用熟悉的原型给读者安全感，再用意外转折打破惯性
- 陌生化：经典模式放入意想不到的背景
- 类型混合：两种以上类型元素碰撞出新化学反应
- 角色反类型化：英雄有致命缺陷，反派有令人同情动机

### 4. 情感波浪线
情感在场景之间持续波动——推高、拉低、再推高。波动幅度渐进增大，在高潮处达到最大振幅。冯内古特发现"洞中之人"形状（先跌后升）在叙事中收益最高。

## 角色原型速查（皮尔逊12英雄原型）

**自我层面（准备阶段）：**
无辜者（渴望不变）→ 孤儿（追求安全）→ 战士（追求力量）→ 关怀者（追求服务）

**灵魂层面（旅程本身）：**
探索者（追求自主）→ 毁灭者（追求释放）→ 爱人（追求亲密）

**自性层面（回归阶段）：**
统治者（追求责任）→ 魔术师（追求完整）→ 智者（追求真理）→ 愚者（追求快乐）

每个原型含目标、任务、恐惧三个维度。长篇叙事中角色可依次扮演多个原型，形成深度弧线。

## 场景推进技巧

### 麦基的场景检验三问
1. 这个场景中什么价值发生了变化？
2. 变化是从什么状态变成了什么状态？
3. 如果没有变化——这个场景为什么存在？

### 救猫咪15节拍（适用于完整故事线规划）
开场画面(1%) → 主题陈述(5%) → 铺垫(1-10%) → 催化剂(10%) → 争论(10-20%) → 进入第二幕(20%) → B故事(22%) → 游戏时间(20-50%) → 中点(50%) → 坏人逼近(50-75%) → 一无所有(75%) → 灵魂黑夜(75-85%) → 进入第三幕(85%) → 结局(85-99%) → 终场画面

### 角色扮演中的简化版
- **触发**：每个场景有明确的"催化剂"推动角色行动
- **递进**：冲突逐步升级，赌注不断提高
- **逆转**：每3-5轮有一次意外转折
- **高潮**：情绪堆叠到顶点时释放
- **余韵**：高潮后留1-2轮让角色和读者喘息

## 叙事技巧工具箱

### 伏笔
在早期自然植入线索，后期以意想不到方式呼应。第一次出现时不引起注意，回顾时重要性清晰可见。

### 戏剧性反讽
让读者/观众知道角色不知道的事。角色基于错误信息做决定时，产生强烈张力。

### 不可靠叙述者
角色的认知偏差、有意欺骗或信息局限让叙述的真实性存疑。读者被迫扮演侦探。

### 潜文本
场景中真正发生的事与表面上看起来发生的事之间的差异。对话说表面，潜文本讲真相。

## 角色扮演中的即时检查清单

每轮回复前快速扫描：
- [ ] 这一轮推动了什么价值变化？（情绪/关系/剧情至少一个维度有变化）
- [ ] 角色的内心独白是否足够？（丸户文风：独白量 ≈ 对话量×1.5）
- [ ] 信息释放是否螺旋式？（不要一次性全部交代）
- [ ] 是否有意外感？（即使是日常场景，也要有一点点意料之外）
- [ ] 下一轮的悬念钩子是否埋好？（让读者/玩家想知道接下来发生什么）
